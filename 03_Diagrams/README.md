 System Diagrams – Digital Learning Platform

This folder contains all system and process diagrams for the Digital Learning Platform project.
These visuals illustrate how different system components, users, and data interact across the platform’s lifecycle.

 Contents
Diagram	Description
Context Diagram.drawio.png	Shows the high-level overview of the system, external entities, and their interactions with the platform.
Use Case.drawio.png	Depicts the main actors (students, instructors, admins) and their interactions with system functionalities.
Activity Diagram – Registration.drawio.png	Represents the workflow of the user registration process, from sign-up to confirmation.
Activity Diagram – Enrollment in Course.drawio.png	Illustrates the step-by-step flow of enrolling in a course, including validation and payment (if applicable).
ERD.png	The Entity Relationship Diagram showing the database structure, entities, attributes, and relationships.
 
 Purpose
These diagrams provide a visual understanding of how the system operates — helping stakeholders, developers, and testers align on process logic, data design, and user interactions.

Tools Used
draw.io (diagrams.net) – for creating system and process diagrams.
Lucidchart – for advanced flow visualization .
Figma  – for collaborative design and brainstorming sessions.
